# Marketplace-Analytics-Engine-End-to-End-Customer-Lifecycle-Churn-Prediction-Pipeline
An enterprise-grade hybrid analytics solution that processes marketplace transactional data using Python, applies advanced machine learning (K-Means Clustering and Random Forest Classifiers) for customer segmentation and churn forecasting, and exposes insights through an interactive Power BI dashboard.
